Introduction

I use nano text editor but you can choose any text editor you want. You also need a C compiler. I'm using GNU C Compiler. I code in my tablet using Termux Linux. if you don't know Linux you must try it. Linux is a free and open source software (FOSS) it has many distros you choose what is working for you.

So let us begin.
